Detailed Description

Base class for collecting validation errors in builder patterns.

This class provides error accumulation functionality for builders that cannot throw exceptions. Builder methods can call AddError() to accumulate validation errors, and CheckErrors() returns all errors at once.

Example usage:

class MyBuilder : public ErrorCollectorBase {
public:
MyBuilder& SetValue(int val) {
if (val < 0) {
return AddError(ErrorKind::kInvalidArgument, "Value must be non-negative");
}
value_ = val;
return *this;
}
Result<MyObject> Build() {
ICEBERG_RETURN_UNEXPECTED(CheckErrors());
return MyObject{value_};
}
private:
int value_ = 0;
};
